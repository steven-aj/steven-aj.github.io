A minimalistic to-do app built with Tauri + Svelte with toggling options between complete & incomplete lists. 

Cliche, perhaps, but to-do lists are a great way to test CRUD operations in experimental stacks. It comes with [Dexie](https://dexie.org) installed - a wrapper for native IndexedDB functions - to persist to-do items, so information isn't lost when the application is closed.
